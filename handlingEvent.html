Handling Event Demo

What is Events?

All the different visitors' actions that a web page can respond to are called events. An event represents the precise moment when something happens.

Example

  • moving a mouse over an element
  • selecting a radio button
  • clicking on an element

            $("document").ready(function(){
                
                $("#textBox").on("click", whenMouseClicked);
                $("#textBox").on("mouseleave", whenMouseRelease);

                function whenMouseClicked (){
                    $("#textBox").html("<h1>Clicked Mouse</h1>")
                }
                function whenMouseRelease (){
                    $("#textBox").html("<h1>Unclicked Mouse<  /h1>")
                }
            });
